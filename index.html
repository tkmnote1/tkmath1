Lập trình C/C++

Con trỏ trong C

Con trỏ là biến dùng để lưu địa chỉ của một biến khác. Đây là kiến thức quan trọng khi học cấp phát động, mảng và danh sách liên kết.

C Pointer

Mảng một chiều

Mảng là cấu trúc lưu nhiều phần tử cùng kiểu dữ liệu trên vùng nhớ liên tiếp. Có thể dùng để lưu danh sách điểm, danh sách số nguyên, dữ liệu thống kê.

Array Basic

Chuỗi ký tự

Trong C, chuỗi là mảng ký tự kết thúc bằng ký tự '\0'. Cần chú ý kích thước mảng khi nhập chuỗi để tránh tràn bộ nhớ.

String C

Cấu trúc dữ liệu

Danh sách liên kết đơn

Danh sách liên kết đơn gồm nhiều node, mỗi node chứa dữ liệu và con trỏ trỏ đến node kế tiếp. Ưu điểm là thêm/xóa linh hoạt, không cần vùng nhớ liên tiếp.

Linked List Node

Stack

Stack hoạt động theo nguyên tắc LIFO: phần tử vào sau sẽ ra trước. Ứng dụng trong kiểm tra dấu ngoặc, khử đệ quy, undo/redo.

Stack LIFO

Queue

Queue hoạt động theo nguyên tắc FIFO: phần tử vào trước sẽ ra trước. Ứng dụng trong hàng đợi xử lý tác vụ, BFS, mô phỏng hệ thống.

Queue FIFO

Toán học

Ma trận

Ma trận là bảng số gồm các hàng và cột. Trong lập trình và machine learning, ma trận dùng để biểu diễn dữ liệu, phép biến đổi và hệ phương trình.

Linear Algebra Matrix

Không gian vector

Không gian vector là tập hợp các vector thỏa mãn phép cộng và nhân vô hướng. Đây là nền tảng của đại số tuyến tính.

Vector Math

DevOps

Git cơ bản

Git giúp quản lý phiên bản mã nguồn. Các lệnh quan trọng gồm git init, git add, git commit, git push và git pull.

Git Version Control

Docker

Docker giúp đóng gói ứng dụng cùng môi trường chạy vào container. Điều này giúp ứng dụng chạy ổn định trên nhiều máy khác nhau.
